# structure_perturbation CLI

Tool for perturbing atomic structures with random or systematic displacements.

The CLI tools have two help layers:

- General help: `python3 toolbox/structure_modelling/structure_perturbation.py -h`
- Tool-specific help: `python3 toolbox/structure_modelling/structure_perturbation.py <tool_name> -h`

## How To Use

```bash
python3 toolbox/structure_modelling/structure_perturbation.py -h
python3 toolbox/structure_modelling/structure_perturbation.py <tool_name> -h
```

## structure_perturbation.py

Tool for perturbing atomic structures with random or systematic displacements.
