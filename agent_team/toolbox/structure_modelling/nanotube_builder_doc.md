# nanotube_builder CLI

Build carbon nanotubes (CNTs) with various chiralities

The CLI tools have two help layers:

- General help: `python3 toolbox/structure_modelling/nanotube_builder.py -h`
- Tool-specific help: `python3 toolbox/structure_modelling/nanotube_builder.py <tool_name> -h`

## How To Use

```bash
python3 toolbox/structure_modelling/nanotube_builder.py -h
python3 toolbox/structure_modelling/nanotube_builder.py <tool_name> -h
```
